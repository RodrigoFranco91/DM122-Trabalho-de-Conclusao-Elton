# DM122-Trabalho-de-Conclusao-Elton
Repositório destinado ao Projeto de Conclusão da Disciplina DM-122 (Prof. Elton) - Desenvolvimento híbrido de aplicativos móveis (Pós).

# PASSOS PARA EXECUTAR O APP:

1- Entrar no diretório appbase (cd appbase);

2- Executar o comando: npm install;

3- Executar o comando: expo start;

4- Ao abrir o navegador clicar em: Run on Android device/emulator.
